## SECTION 9: FAQ

**Q: Is this really free?**  
A: Yes. The core ARK software is MIT licensed open source. Paid tiers are for convenience (USB kits, managed updates), not features.

**Q: Do I need to be technical?**  
A: Basic command-line comfort helps. If you've installed Linux or used Docker, you're fine. Detailed docs included.

**Q: What hardware do I need?**  
A: Any x86-64 machine: old laptop, NUC, mini-PC, or VM. 4GB RAM minimum, 8GB recommended.

**Q: Does it work on Raspberry Pi?**  
A: Not yet. ARM support coming in v2.0 (Q3 2026).

**Q: How is this different from Umbrel?**  
A: ARK includes AI (Ollama), more media options, and is nomad-optimized (offline-first, power-aware). Umbrel is Bitcoin-focused.

**Q: Can I access this remotely?**  
A: Use Tailscale or Cloudflare Tunnel for secure remote access. Guides included.

**Q: What if I get stuck?**  
A: 42-file documentation, community Discord, GitHub issues. Paid tiers include email support.
